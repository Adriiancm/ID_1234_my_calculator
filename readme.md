# Calculadora Web

Esta es mi calculadora web de la práctica ID_1234.

## Autor
Adrian Calle Martin

## Funcionalidad
- Calcula el cubo de un número (x³)
- Calcula la cuarta potencia de un número (x⁴)

## Cómo usar
1. Abrir index.html en un navegador
2. Introducir un número
3. Pulsar el botón x³ o x⁴ para obtener el resultado
